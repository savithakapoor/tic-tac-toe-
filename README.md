# Tic Tac Toe Game in C++

## Project Description
This is a console-based Tic Tac Toe game developed using C++.  
The project demonstrates core programming concepts such as:

- Loops
- Arrays
- Functions
- Conditional Statements
- Game Logic

## Features
- Dynamic game board display
- Two-player gameplay
- Win and draw detection
- Replay functionality
- Input validation

## Technologies Used
- C++
- Console Application

## How to Run

1. Compile the program:
g++ tic_tac_toe.cpp -o game

2. Run the executable:
./game

## Sample Output

1 | 2 | 3
---|---|---
4 | 5 | 6
---|---|---
7 | 8 | 9

Player X, enter position (1-9): 5

- Arrays in C++
- Nested loops

- Game development basics
